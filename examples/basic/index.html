JS3 DEMO - Basic Operations

JavaScript Style Sheet (JS3) engine puts CSS inside JavaScript and rest is the power of JavaScript and CSS itself that does all the magic.

This demo sets up the environment with JS3 engine and few JavaScript Style Sheets (.js3 files) to make the playground ready for you to play around with JS3 awesomeness.

Open console window and try below suggested statements to see the impact in real time. Go in any order and see what happens on the page. Hover on statements to see details, if any. When wants to start a fresh, just reload the page and start from any other place this time. Enjoy!!!

  • JS3.normal.styles.borders.off()
  • JS3.normal.styles.borders.on()
  • JS3.normal.rules.allMargin(50)
  • JS3.normal.sel.Everywhere.styles.remove(JS3.normal.styles.spacing)
  • JS3.normal.sel.Everywhere.styles.add(JS3.normal.styles.spacing)
  • JS3.normal.styles.code.rules.add(JS3.normal.rules.CodeColor)
  • JS3.normal.styles.code.rules.remove(JS3.normal.rules.CodeColor)
  • JS3.normal.vars.unit('em')
  • JS3.normal.vars.unit('px')
  • JS3.normal.vars.lightColor('yellow')
  • JS3.normal.styles.borders.rules()[2].off()
  • JS3.normal.styles.borders.rules()[2].on()
  • JS3.widgets.load()
  • JS3.widgets.unload()
  • JS3.widgets.load(JS3.normal.sel.WidgetsContainer.raw.value())
  • JS3.suspendUpdates()
  • JS3.normal.rules.allPadding.reset('xLarge')
  • JS3.resumeUpdates()
  • JS3.normal.rules.allPadding.reset()
